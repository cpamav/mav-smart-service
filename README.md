# mav-smart-service
smart service system

V. 2.0
หน้านอกเวลาทำการ
  -มีปุ่มติดตามงาน ปุ่มล็อกอินสำหรับเจ้าหน้าที่
  -เวลาทำการเว็บ จ-ศ 08:30-16:30 น.
  -วันหยุดราชการจะเป็นนอกเวลาทำการ(ดึงจาก Google Calendar)
หน้าแอดมิน
  -เพิ่มปุ่ม เปิด-ปิด หน้าเว็บนอกเวลาทำการ
แอดมิน-สตาฟ
  -หน้าแอดมินและสตาฟปรับปุ่ม notif ให้อัปเดทงานโดยไม่ต้องรีเฟรชหน้า เมื่อแอดมิน assign งานให้เจ้าหน้าที่ หรือเจ้าหน้าที่กดรับงานแล้วหน้าการ์ดงานจะอัปเดทให้เองโดยไม่ต้องรีเฟรชหน้า
  -รีเฟรชหน้าไม่ล็อกเอาท์แล้ว
อัปโหลดไฟล์ (ขอรับบริการ)
 -อัปโหลดไฟล์ได้หลายไฟล์ เพิ่ม และ ลบ ไฟล์ได้
  -กดดูภาพที่จะอัปโหลดได้
ไฟล์ที่ถูกอัปโหลด
  -เมื่อมีไฟล์อัปโหลดจะสร้างไดร์ฟแยกตามไอดีงาน
  -ลิงก์ที่เก็บในชีทเป็นลิงก์โฟลเดอร์งาน
  -หน้าการ์ดงานในแอปแสดงลิงก์โฟลเดอร์ไฟล์แนบของงานนั้น
